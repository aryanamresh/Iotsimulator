Requirements :- Python3, flask, numpy for backend
                Node Package Manager (npm) for frontend

flask can be installed by command : pip install flask-restful

For running the web app :-
Step-1 : Install all the packages required for front end. Run the following command on node.js command prompt:-
	 cd WebApp_SRIB
	 npm install

Step-2 : Start the backend server. Run the command on the command prompt:-
	 cd WebApp_SRIB/backend
	 python App.py

Step-3 : Start the web app frontend. Run the command on the node.js command prompt:-
	 cd WebApp_SRIB
	 npm start
